Microservices in Python


Overview
========


Requirements
===========

    * Python 2.7
    * Works on Linux, Windows, Mac OSX and (quite possibly) BSD.

Install/Run
===========

    Commands:$

    * virtualenv venv --python=python2.7

    * . venv/bin/activate

    * pip install -r requirements.txt

    * python run.py

    Url route:

    * http://localhost:5000/api/user/
