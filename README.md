Microservices in Python


Overview
========


Requirements
===========

    * Python 2.7

    * Works on Linux

Install/Run
===========

    Commands:$

    * virtualenv venv --python=python2.7

    * . venv/bin/activate

    * pip install -r requirements.txt

    * python db_create.py

    * python db_migrate.py

    * python run.py

    Url route:

    * http://localhost:5000/api/get/

    * http://localhost:5000/api/post/
    
    * http://localhost:5000/api/update/
